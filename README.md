# 🎓 CSE Career Care Portal

The **CSE Career Care Portal** is a full-stack web application developed to centralize and manage all placement, internship, research, and alumni-related activities within a Computer Science department.

It provides a **secure, role-based system** where students can access important career-related information, while administrators manage and update the data efficiently.

## 🎯 Objectives
- Simplify placement and internship processes
- Improve communication between students and administration
- Provide a centralized platform for career resources
- Maintain structured alumni data for networking

---

- Resume upload and review system

---

## 🚀 Features

### 👨‍🎓 Student Interface
- View placement opportunities and company details
- Access winter, summer, and M.Tech internship information
- Explore research projects and faculty details
- Browse alumni directory with filters (company, location, year)
- Access FAQs and contact information

### 🛠️ Admin Interface
- Add and manage student accounts
- Perform CRUD operations on:
  - Placements
  - Internships
  - Research projects
  - Alumni data
- Secure dashboard for managing all portal data

---

## 🔐 Authentication & Security
- JWT-based authentication
- Role-based access control (Student / Admin)
- Password hashing using bcrypt
- No public registration (students are added by admin only)
- Admin accounts created securely via backend scripts
